# MineTrigger

**Minecraft Fabric Mod (1.21.1 / Fabric API 0.116.11 / Java 21)**

ワールドトリガーの世界観をMinecraftに再現するFabric Modです。
トリオンゲージ・トリガー（武器）・多彩な戦術をMinecraft上で体験できます。

---

## 概要

- **トリオン**: 全ての行動の原動力となるエネルギー。残量がなければトリガーは発動できません。
- **トリガー**: 攻撃・防御・移動などさまざまな効果を持つ武器アイテム。手に持って右クリックで発動します。
- **クールダウン**: 各トリガーには発動後の待機時間があります。
- **ロードアウト**: 左手・右手・オプションスロットにトリガーをセットして戦います。

### トリオンゲージ

- 攻撃を受けるとトリオンが減少（体力ダメージ）
- 武器を使用するとトリオンを消費（技コスト）
- バッグワーム装備中は継続消費
- トリオンは戦闘中に回復しない
- トリオンがゼロになるとベイルアウト（戦闘離脱）

### トリオン量の計算

装備武器が少ないほどトリオン総量が多くなり、攻撃力も上がります。

```
実効トリオン総量 = 基礎トリオン量 − Σ(装備武器のトリオン装備コスト)
攻撃力倍率 = 実効トリオン総量 × attack_power_multiplier
```

## 操作説明

| 操作 | 効果 |
|------|------|
| 右クリック | 手に持ったトリガーの **通常技** を発動 |
| Shift + 右クリック | 手に持ったトリガーの **特殊技** を発動 |
| 数字キー 1〜9 | ホットバーのトリガーを選択 |
| E キー | インベントリを開く（オプショントリガーの設定） |

> ヒントメッセージはアクションバー（画面中央下）に表示されます。

### ロードアウトの組み方

| ホットバースロット | 用途 |
|---|---|
| 0〜3 | メインハンド用 |
| 4 | オプション用（Qキーで発動） |
| 5〜8 | オフハンド用 |

## コマンド一覧

### トリオン操作（/trion）

OP権限（パーミッションレベル2以上）が必要です。

| コマンド | 説明 |
|---------|------|
| `/trion heal` | 自分のトリオンを最大値まで回復 |
| `/trion heal <player>` | 指定プレイヤーのトリオンを全回復 |
| `/trion set <量>` | 自分のトリオンを指定値に設定 |
| `/trion set <量> <player>` | 指定プレイヤーのトリオンをセット |
| `/trion add <量>` | 自分のトリオンに加算（負の値で減算も可） |
| `/trion add <量> <player>` | 指定プレイヤーのトリオンに加算 |
| `/trion get` | 自分の現在トリオン残量を表示 |
| `/trion get <player>` | 指定プレイヤーのトリオン残量を表示 |

使用例：
```
/trion heal
/trion set 50
/trion add 30 @a
/trion get
```

## オプショントリガーの設定方法

1. **E キー**でインベントリを開きます。
2. インベントリ画面の**左側パネル**（オプショントリガースロット）が表示されます。
3. オプション系のトリガーアイテムをそのスロットに**ドラッグ&ドロップ**します。
4. 発動するには、対応する**数字キー**を押してください（またはQキー）。

> オプショントリガーは通常の手持ち武器とは独立して発動できます。

## config.yml について

`config/minetrigger/config.yml` に各トリガーのパラメータが記述されています。
サーバー管理者はここを編集してバランス調整ができます。

### 主なパラメータ

| パラメータ | 説明 |
|-----------|------|
| `damage` | 基礎ダメージ値 |
| `range` | 射程距離（ブロック単位） |
| `speed` | 弾速（倍率） |
| `cooldownTicks` | クールダウン（tick単位。20tick = 1秒） |
| `trionUse` | 1回の発動で消費するトリオン量 |
| `trionEquipCost` | 装備中に差し引かれるトリオンコスト |
| `splashRadius` | 爆発範囲（ブロック） |
| `pelletCount` | 散弾数（ショットガン系） |
| `trackingSpeed` | ホーミング強度（0.0〜1.0） |
| `damageReduction` | シールドのダメージ軽減率（0.0〜1.0） |
| `catapultVelocity` | エスクードカタパルトの上方向速度 |
| `wireLength` | スパイダーのワイヤー最大長 |

### 設定例

```yaml
weapons:
  handgun:
    damage: 6
    range: 30
    speed: 2.0
    cooldownTicks: 8
    trionUse: 3
    trionEquipCost: 2
  ibis:
    damage: 20
    range: 80
    speed: 1.5
    cooldownTicks: 40
    trionUse: 15
    trionEquipCost: 10
    splashRadius: 4.0
  escudo:
    damageReduction: 0.6
    catapultVelocity: 2.0
    cooldownTicks: 20
    trionUse: 8
  spider:
    wireLength: 20.0
    speed: 1.5
    cooldownTicks: 15
    trionUse: 4
```

## インストール

1. [Fabric Installer](https://fabricmc.net/use/) でFabricをインストール
2. `mods/` フォルダに本Modの `.jar` ファイルと Fabric API を配置
3. 起動後に `config/minetrigger/config.yml` が自動生成されます
