# MineTrigger

ワールドトリガーの世界観をMinecraftに再現するFabric Modです。

## 概要

トリオンゲージ・トリガー（武器）・ランク戦をMinecraft上で実装します。
原作の「少ないトリオンでいかに戦うか」という駆け引きをゲームプレイに反映しています。

## 動作環境

- Minecraft Java Edition（最新安定版）
- Fabric Loader
- Fabric API

## インストール

1. [Fabric Installer](https://fabricmc.net/use/) でFabricをインストール
2. `mods/` フォルダに本Modの `.jar` ファイルを配置
3. 起動後に `config/worldtrigger/config.yml` が自動生成されます

## 基本仕様

### トリオンゲージ

通常の体力の代わりに**トリオンゲージ**を使用します。

- 攻撃を受けるとトリオンが減少（体力ダメージ）
- 武器を使用するとトリオンを消費（技コスト）
- バッグワーム装備中は継続消費
- トリオンは戦闘中に回復しない
- トリオンがゼロになるとベイルアウト（戦闘離脱）

### トリオン量の計算

装備武器が少ないほどトリオン総量が多くなり、攻撃力も上がります。

```
実効トリオン総量 = 基礎トリオン量 − Σ(装備武器のトリオン装備コスト)
攻撃力倍率 = 実効トリオン総量 × attack_power_multiplier
```

### キャラクターステータス

| パラメータ | デフォルト値 |
|---|---|
| 移動速度 | バニラの1.5倍 |
| ジャンプ力 | バニラの2.0倍 |
| 基礎トリオン量 | 100 |

## 操作方法

| 操作 | 動作 |
|---|---|
| 左クリック | 左手の武器を通常発動 |
| 右クリック | 右手の武器を通常発動 |
| シフト＋左クリック | 左手の武器の特殊技発動 |
| シフト＋右クリック | 右手の武器の特殊技発動 |
| スクロール | ホットバーのスロット切り替え |
| Qキー | 選択中スロットのオプショントリガー使用 |

### ロードアウト

- 最大8スロット（メイン4＋サブ4）
- 同時使用は2つまで（メイン＋サブ）
- **片手武器**：弧月・スコーピオン・レイガスト・ハンドガン・シューター弾 → 2個まで同時装備可
- **両手武器**：アサルトライフル・スナイパー系など → 1個まで
- 銃型トリガーと弾丸トリガーは別スロット扱い

## トリガー（武器）一覧

### アタッカー系

| 武器名 | 通常発動 | 特殊技 |
|---|---|---|
| 弧月 | 斬撃 | 旋空（射程延長）／幻踊（刀身変形） |
| スコーピオン | 斬撃 | 形状切り替え（ナイフ／長剣／二刀／槍） |
| レイガスト | 斬撃 | 盾モード切り替え |

### ガンナー系

| 武器名 | 特徴 |
|---|---|
| ハンドガン | 片手運用可。コンパクト |
| アサルトライフル | 連射型。最も使用率が高い |
| ショットガン | 近距離散弾。近距離時に威力↑ |
| ガトリング | 高威力連射。トリオン消費大 |
| グレネードガン | メテオラを放物線軌道で発射 |

### シューター系（弾丸トリガー）

| 武器名 | 特徴 |
|---|---|
| アステロイド | 直進する標準弾 |
| メテオラ | 炸裂弾。地雷設置も可能（シフト＋クリック） |
| ハウンド | 追尾弾。目標を自動追跡 |
| バイパー | 事前にノード（最大5つ）を設置してから発射する軌道弾 |
| レッドバレット | シールドを無視してスロウネスを付与する鉛弾 |

#### バイパーの操作

1. シフト＋クリックで空中にノードを設置（最大5つ）
2. シフトを離してクリックでノードを経由するビームを発射
3. ノードは一定時間で自動消去

### スナイパー系

| 武器名 | 特徴 |
|---|---|
| イーグレット | バランス型。トリオン量で射程↑ |
| ライトニング | 高弾速型。トリオン量で弾速↑ |
| アイビス | 高威力型。トリオン量で威力↑・シールド貫通率↑ |

### 防御系

| 武器名 | 特徴 |
|---|---|
| シールド | 全員共通の防御バリア |
| エスクード | 地面に永続する壁を設置。真下設置でジャンプ力大幅↑（カタパルト） |

### オプション系（Qキーで使用）

| 武器名 | 動作 |
|---|---|
| バッグワーム | レーダー非表示のオン/オフ切り替え（継続トリオン消費） |
| グラスホッパー | 向いた方向の1m先に足場を設置（5秒で消滅） |
| スパイダー | ワイヤーを展開。範囲内の敵にスロウネス付与 |
| スイッチボックス | トラップを設置し、任意のタイミングで起動 |

## 設定ファイル

`config/worldtrigger/config.yml` で各種パラメータを変更できます。

```yaml
player:
  base_trion: 100               # 基礎トリオン量
  move_speed: 1.5               # 移動速度倍率
  jump_boost: 2.0               # ジャンプ力倍率
  attack_power_multiplier: 0.5  # トリオン→攻撃力の変換係数

weapons:
  kogetsu:
    trion_equip_cost: 5         # 装備時のトリオン消費
    damage: 8                   # 基礎ダメージ
    range: 3.0                  # 射程
    cooldown_ticks: 10          # クールダウン（tick単位）
  asteroid:
    trion_equip_cost: 6
    damage: 7
    range: 20.0
    cooldown_ticks: 15
  # ...以下全武器

options:
  grasshopper:
    max_count: 4                # 同時設置上限
    duration_seconds: 5         # 足場の持続時間
    place_distance: 1           # 設置距離（m）
  viper:
    max_nodes: 5                # ノード最大数
    node_duration_seconds: 8    # ノード持続時間
  bagworm:
    trion_drain_per_second: 2   # 継続消費量

game:
  trion_regen: false            # 戦闘中のトリオン回復なし
  critical_rate_base: 0.1       # 基礎クリティカル率
  critical_damage_multiplier: 2.0 # クリティカルダメージ倍率
```
